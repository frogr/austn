So I just set up [Hermes Agent](https://github.com/hermes-agent/hermes) — it's an open-source AI agent that sits in my Discord server and has access to my dev machine. I'm writing this post by DMing it on Discord right now, which still kind of blows my mind.

## Setup took maybe 10 minutes

Honestly I expected more friction. You install Hermes, go through the config, and it mostly just works out of the box. I had to refresh my Claude Code OAuth login and generate a Discord bot token, both of which were trivial. That was it. Ten minutes, tops.

## The tool list is absurd

When the bot joined my server it registered over a hundred tools. File editing, terminal access, git, web search, cron jobs, code search — the whole kitchen sink. It's like having a full dev environment inside a chat window.

The part that really got me was realizing I could DM this thing and it would run commands on my machine. Read files, edit code, run builds, search through repos. All from the Discord app I already have open all day anyway.

## Open source makes all the difference

I've been deep in Claude Code lately and I love it. But Hermes being open source changes the equation. I can actually dig in, contribute, customize how the agent behaves, add tools for my specific workflow. Nothing is a black box.

I really do think OSS is the future for AI tooling. These agent frameworks are getting ridiculously capable, and being able to fork, extend, and own your setup matters a lot more than people realize right now.

## This post exists because I talked to a bot

I want to be clear about what happened here: I opened Discord, sent some voice messages and texts to my bot about what I wanted to say, and it turned that into a markdown file sitting in my blog's content directory, ready to deploy. It already knew the whole workflow — where files go, how frontmatter works, all of it.

That's what gets me excited about this stuff. Not the AI writing for me, but the friction just disappearing. I had a thought, I talked about it, and now it's a post. No editor, no terminal, no remembering file paths. Just a conversation.
